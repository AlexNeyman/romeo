## Installation

If [available in Hex](https://hex.pm/docs/publish), the package can be installed as:

1. Add romeo to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:romeo, "~> 0.0.1"}]
end
```

2. Ensure romeo is started before your application:

```elixir
def application do
```
